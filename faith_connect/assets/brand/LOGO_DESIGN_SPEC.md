# FaithConnect Logo Design Specification

## Design Concept: "Interconnected Faith"

### Core Idea
The logo represents **unity in diversity** - multiple faith symbols interconnected, symbolizing how FaithConnect brings people of different spiritual paths together.

### Visual Elements

#### 1. **Central Hub** (Primary Focus)
- **Shape**: Circle with infinity symbol inside
- **Color**: Purple gradient (matching app theme: #9D8BF5 â†’ #8B7CF0 â†’ #7B6FE8)

#### 2. **Faith Symbols** (Positioned Around Center)
- **Om Symbol** (Hinduism): Top-left quadrant, stylized
- **Crescent & Star** (Islam): Top-right quadrant
- **Cross** (Christianity): Center-bottom

### Color Palette

#### Primary Colors (Spiritual Purple)
- **Light**: #9D8BF5 (Lavender)
- **Mid**: #8B7CF0 (Violet)
- **Dark**: #7B6FE8 (Deep Purple)

#### Accent Colors (Divine Gold)
- **Light**: #FFD700 (Gold)
- **Dark**: #FFA500 (Amber)

### Typography (For Text Logo)

When used with text, recommend:
- **Font**: Modern, clean sans-serif (Montserrat, Poppins, or Inter)
- **Style**: Bold, slightly rounded
- **Color**: Deep Purple (#7B6FE8) or White
- **Spacing**: Letter-spacing: 0.05em for elegance

### Logo Variations

#### 1. **Full Logo** (Icon + Text)
```
[Icon] FaithConnect
```

#### 2. **Icon Only** (App Icon)
- Square format (512x512px minimum)
- Rounded corners option (iOS/Android)

#### 3. **Monochrome** (For single-color use)
- All elements in single color (purple or gold)
- Maintains recognizability

### Usage Guidelines

#### Minimum Sizes
- **App Icon**: 512x512px (iOS), 1024x1024px (Android)
- **Logo with Text**: 256px height minimum
- **Favicon**: 32x32px (simplified version)

#### Clear Space
- **Rule**: Equal to half the icon width on all sides
- Example: For 512px icon, clear space = 256px

#### Backgrounds
- **Primary**: White or light backgrounds
- **Secondary**: Purple gradient backgrounds
- **Dark Mode**: White/gold version

### Implementation Files

1. **SVG Logo**: `faithconnect_logo_new.svg` (Scalable vector)
2. **PNG Exports**: Generate at multiple sizes
   - 512x512px (standard app icon)
   - 1024x1024px (high-res)
   - 256x256px (medium)
   - 128x128px (small)
   - 64x64px (favicon)
